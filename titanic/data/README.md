# Data

This directory is gitignored. Download the competition data with:

```bash
cd ..
python setup_kaggle.py
```

Or manually:

```bash
kaggle competitions download -c titanic -p data/raw/
unzip data/raw/titanic.zip -d data/raw/
```

## Expected structure

```
data/
├── raw/
│   ├── train.csv
│   ├── test.csv
│   └── gender_submission.csv
└── processed/        ← generated by notebooks
```
